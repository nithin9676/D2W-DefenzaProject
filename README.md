# 🚗 D2W Defenza Backend Project

This is a Spring Boot backend application for managing warranty and role inventory system for D2W Defenza Paint Protection Film (PPF).

---

## 📌 Features

- 🔐 Admin Login
- 🧾 Warranty Registration
- 📦 Role Inventory Management
- 📊 Dashboard Counts (Total, Available, Used, Damaged)
- 🔎 Filter Roles by Status

---

## 🛠️ Technologies Used

- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- PostgreSQL
- Hibernate

---

## 📂 Project Structure

- `controller` → API endpoints  
- `service` → Business logic  
- `entity` → Database models  
- `repository/dao` → Database operations
